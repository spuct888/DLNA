# DLNA Receiver

DLNA投屏接收器 for Android TV

## 功能特点

- DLNA投屏接收功能
- 支持视频旋转（按菜单键）
- 支持视频画面缩放（按上下键）
- 支持主流视频格式播放

## 使用说明

1. 安装APK到Android TV设备
2. 打开应用，等待投屏连接
3. 在手机上打开视频应用，点击投屏按钮
4. 选择本设备进行投屏

### 遥控器操作

- **确认键**：暂停/播放切换（V9新增）
- **菜单键**：视频旋转90度
- **上键**：视频画面放大
- **下键**：视频画面缩小
- **左键**：快退10秒
- **右键**：快进10秒

## 构建

```bash
./gradlew build
```
